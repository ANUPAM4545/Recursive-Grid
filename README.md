# The Recursive Grid

A frontend technical evaluation project built with Next.js and Tailwind CSS.

## Features

- **3x3 Grid**: Interactive grid with custom logic.
- **Ripple Rules**:
    - Numbers divisible by 3 decrement their right neighbor.
    - Numbers divisible by 5 increment their bottom neighbor.
- **Locking Mechanism**: Boxes lock and turn red when they reach value 15.
- **Animations**: Smooth hover, tap, and pop effects using Tailwind CSS v4.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.
